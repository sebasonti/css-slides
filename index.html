¿Qué es CSS?

Cascading Style Sheets (Hojas de estilo en cascada)

HTML se utiliza para definir la estructura y la semántica del contenido, CSS se usa para darle estilo y posicionarlo visualmente

cambiar la fuente, el color, el tamaño y el espaciado del contenido, para formar multiples columnas, añadir animaciones y otros elementos decorativos.

Sintaxis de CSS

La propiedad es un identificador, un nombre leíble por humanos, que define qué característica es considerada.

El valor describe como las características deben ser manejadas por el motor. Cada propiedad tiene un conjunto de valores válidos

CSS inline

El CSS inline se utiliza para dar estilo a un elemento HTML específico.

<h1 style="padding:30px;">Club Chicas Programadoras</h1>

CSS interno

El CSS interno o incrustado requiere que añadas la etiqueta <style> en la sección <head> de tu documento HTML.

<style type="text/css">

body {

background-color: blue;

}

</style>

CSS externo

Con el CSS externo, enlazarás tus páginas web a un archivo .css externo, que puede ser creado con cualquier editor de texto de tu dispositivo

body {

background-color: rebeccapurple;

}


En la sección <head> de tu documento HTML, añade una referencia a tu archivo .css externo justo después de la etiqueta <title>

<link rel="stylesheet" type="text/css" href="style.css" />

Selectores CSS

Básicos

Los selectores en CSS definen qué elementos serán afectados por las reglas de CSS.

* - Universal: Selecciona todos los elementos

div - De tipo: Selecciona todos los elementos de ese tipo

.main-title - Clase: Selecciona elementos que tengan esa clase

#username - Id: Selecciona elementos que tengan ese id (debería ser único)

Agrupados

h1, h2, h3 - Selecciona todos los elementos que cumplan los selectores separados por comas

Selectores CSS

Combinados

div a - Descendiente: Selecciona elementos que son descendientes del primer elemento (todos los a dentro de un div)

div > a - Hijo directo: Selecciona elementos que son hijos directos del primer elemento

div ~ a - Hermanos siguientes: Selecciona elementos que son hermanos del primer elemento y que aparecen después

div + a - Hermano adyacente: Selecciona el primer elemento que es hermano del primer elemento y que aparece después

Selectores CSS

Atributos

[atributo] - Tiene: Selecciona elementos que tienen ese atributo

[atributo="valor"] - Igual: Selecciona elementos que tienen ese atributo con exactamente ese valor

[atributo^="valor"] - Comienza con: Selecciona elementos que tienen ese atributo y comienza con ese valor

[atributo$="valor"] - Termina con: Selecciona elementos que tienen ese atributo y termina con ese valor

[atributo*="valor"] - Contiene: Selecciona elementos que tienen ese atributo y contiene ese valor

Tipos de unidades

Unidades absolutas

La más utilizada es el pixel: 1px

Unidades relativas

Las unidades relativas dependen de algún otro factor (resolución, tamaño de letra, etc...).

em: Tamaño de fuente en el contexto

rem: Tamaño de fuente en la raíz

%: Relativo al elemento padre

Tipos de unidades

:root {

}

h1 {
font-size: 2rem; /* El doble del tamaño base: 44px */
}

h2 {
font-size: 1rem; /* El mismo tamaño base: 22px */
}

Tipos de unidades

Unidades flexibles

Con estas unidades podemos hacer referencia a un porcentaje concreto del tamaño específico que tengamos en la ventana del navegador, independientemente de si es redimensionado o no.

vw: 1vw = 1% del ancho del navegador

vh: 1vh = 1% del alto del navegador

vmin: 1vmin = 1% del ancho o el alto del navegador, el que sea menor

vmax: 1vmax = 1% del ancho o el alto del navegador, el que sea mayor

Modelo de Caja

Cuando hablamos del modelo de cajas en CSS, estamos haciendo referencia a un sistema que tiene el navegador de graficar los elemento HTML como cajas.

Dimensiones

Las propiedades que modifican las dimensiones de un elemento son:

height: modifica la altura del elemento

width: modifica el ancho del elemento

margin: es un abreviado de margin-top margin-right margin-bottom margin-left

padding: es un abreviado de padding-top padding-right padding-bottom padding-left

Como buena práctica se recomienda configurar la propiedad box-sizing: border-box; para que las propiedades height y width coincidan con la suma del contenido, el padding y el borde.

Manejo de Colores

Los colores pueden ser declarados de distintas maneras.

Palabras clase

Nombres de colores son reconocidos por CSS, como ser: red, yellow, blue, tomato, salmon, beige

rgb: Red Green Blue

Existe una función en CSS que permite definir un color por sus valor Red-Green-Blue, que van del 0 al 255. Por ejemplo, un gris sería rgb(100, 100, 100)

Hay una variación de la función rgb que incluye un parámetro que permite definir un grado de transparencia: rgba(100, 100, 100, 50%) - gris transparente al 50%

Hexadecimal

Los valores RGB, expresados en números decimales, también puede ser expresado en número hexadecimales de esta manera: #4ac1d2, son dos dígitos para R, dos para G y dos para B

Manejo de Colores

hsl: Hue Saturation Lightness

Es otra función el CSS que recibe otros parámetros.

Hue: Es el grado de 0 a 360 del círculo de colores. 0 es rojo, 120 es verde y 240 es azul

Saturation: es el porcentaje de saturación en una escala de grises, 0% es gris y 100% es el color puro

Lightness: es el porcentaje de brillo, 0% es negro total, 100% es blanco y 50% es el color puro

Ejemplo: hsl(240, 100%, 50%) - azul

Manejo de fuente

La propiedad font es una abreviación de muchas propiedades que modifican la fuente de texto.

font-size (obligatorio): Define la altura del texto.

font-style: Define si el texto debe ser normal, italic, oblique según la familia de fuente

font-weight: Define el grosor de los trazos del texto, según la familia fuente; como normal, bold.

font-stretch: Define el grado de anchos y separaciones entre letras del texto, como normal, condensed o expanded

line-height: Define la altura de las lineas de un text, es decir, la separación entre ellas

font-family (obligatorio): Define la familia de la fuente. Un buen repositorio de fuentes es Google Fonts

Bordes

Los bordes de los elementos están definidos por la propiedad border.

border es una abreviación de:

border-color: El color del borde

border-style: El estilo del border, que puede ser: dotted, dashed, solid, double, etc

border-color: Define el color del borde

Cada una de las propiedades mencionadas son a su vez abreviaciones de las propiedades correspondiendes a top, right, bottom y left.

border-radius: Modifica las esquinas de un elemento de forma redondeada. Es una abreviación de top-left, top-right, bottom-right, bottom-left.
